# Three.js Journey 🌐

A collection of Three.js and WebGL experiments built while learning 3D graphics for the browser.

## What's Inside
- Basic 3D scenes and animations
- Shader experiments (GLSL)
- Orbit controls and camera rigs
- Geometry and material playground
- Particle systems

## Tech Stack
- Three.js
- WebGL / GLSL shaders
- Vite (bundler)

## Getting Started

```bash
git clone https://github.com/RiyanMujtaba/threejs-journey
cd threejs-journey
npm install
npm run dev
```
